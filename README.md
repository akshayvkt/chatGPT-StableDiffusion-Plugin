# ChatGPT Stable Diffusion Plugin

## Generate and Edit Images within ChatGPT UI Using This Plugin

**Requirements:** Access to ChatGPT plugins, and a Replicate API key.

With the **ChatGPT Stable Diffusion Plugin**, you can generate and edit images through natural language prompting. Images are generated by making API calls to Replicate's Stable Diffusion model. Since ChatGPT is context-aware, editing the generated images is much easier.
